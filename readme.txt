Easily add styles for specific browsers and rendering engines, from the dashboard!

== Description ==

This plugin sniffs users agents, and provides a set of theme options for a user to put browser-specific CSS styles. It's very easy to use, very simple, and very powerful.

== Installation ==

1. Upload `CSSniffer.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click of 'CSSniffer' in the Appearance tab.
4. Hack some CSS like no tomorrow!

== Frequently Asked Questions ==

= What browsers can I target? =

Right now, you can specifiy CSS rules for Internet Explorer, Firefox, Webkit, Opera, and iPhone.
